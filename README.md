# Netflix-Data-Analysis-PowerBI
An interactive Power BI dashboard analyzing Netflix's dataset (8,800+ titles) to uncover insights into content distribution, rating trends, and top global markets.
# Project Overview
This project analyzes the Netflix dataset to visualize content strategy, focusing on Movies vs. TV Shows, geographical distribution, and audience ratings.
# Key Features & Insights
1.Total Content: Analyzed 8,807 unique titles.

2.Content Mix: Discovered that Movies make up ~70% of the library compared to ~30% for TV Shows.

3.Top Markets: Identified the United States and India as the leading content producers.

4.Rating Analysis: Visualized audience reach using a Treemap based on content ratings like TV-MA and TV-14.
# Tools Used
1. Power BI Desktop: For Data Visualization and Dashboarding.
2. Power Query: For Data Cleaning and Transformation (Handling nulls in Country/Rating).
3. SQL : for data handling
